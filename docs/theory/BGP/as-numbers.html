AS Numbers Overview

ทำความเข้าใจเกี่ยวกับ Autonomous System Numbers และการใช้งาน

Private AS Range

16-bit: 64512-65534
32-bit: 4200000000-4294967294
ใช้ภายในองค์กรเท่านั้น
ไม่สามารถประกาศบนอินเทอร์เน็ต
ใช้ใน BGP Confederation

Public AS Range

16-bit: 1-64495
32-bit: 131072-4199999999
ต้องลงทะเบียนกับ Internet Registry
ใช้เชื่อมต่อกับอินเทอร์เน็ต
ประกาศบนอินเทอร์เน็ตได้

Reserved AS Numbers

0, 65535: Reserved
64496-64511: Documentation
23456: AS_TRANS
ใช้สำหรับเอกสารและการทดสอบ
AS_TRANS ใช้สำหรับ 16-bit to 32-bit transition

การใช้งาน Private AS Numbers

ใช้ในเครือข่ายภายในองค์กร
เหมาะสำหรับ BGP Confederation
ต้องทำ AS number stripping เมื่อเชื่อมต่อกับ Public Internet
หมายเหตุ: เมื่อใช้ Private AS Numbers ต้องระวังไม่ให้หลุดไปประกาศบน Public Internet
